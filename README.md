## Features

- **DIRTY TITIAN Consensus Engine**: Cross-verifies data from 4+ high-accuracy APIs for 100% location precision.
- **Multithreaded DIRTY TITIAN Threader**: Extreme-performance parallel tracking for massive batch operations.
- **Deep Intelligence Dash**: Real-time insights into Carrier, Connection Type, and ASN metadata.
- **Risk Assessment**: Advanced detection of VPNs, Proxies, Crawlers, and Datacenters.
- **Reverse DNS & WHOIS**: Instant hostname resolution and deep organizational ownership data.

## Installation

- `apt update`
- `apt install git curl jq whois dnsutils iputils-ping -y`
- `git clone git://github.com/Dirty-X/dirty-track.ip.git dirty-track`
- `cd dirty-track`

### > Run: `bash dirty-track.ip`
